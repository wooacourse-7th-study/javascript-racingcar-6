## 🏎️ 자동차 경주

---

### 💡기능 정의

- 경주할 자동차 이름을 입력 받는다. (이름은 쉼표(,) 기준으로 구분)

  - 사용자 입력시 (체크 리스트)
  - ✔️ 한글자 이상 입력
  - ✔️ 이름 5글자 이하 (공백 및 스페이스도 길이에 포함)
  - ✔️ 이름 중복 확인
  - ❌ 그 외 입력시 예외 처리

- 시도할 횟수는 몇 회인지 입력 받는다.

  - 사용자 입력시 (체크 리스트)
  - ✔️ 오직 숫자만
  - ✔️ 0 이상 입력
  - ❌ 그 외 입력시 예외 처리

- "실행 결과"라는 텍스트와 함께
- 자동차 전진을 카운트하여 결과창에 보여준다.
- 최종 우승자 출력
